## Overview
Foodoscope is a data analytics project that recommends vegetarian meals by
balancing nutritional health, cultural preference, and flavor profile.
The recommendations are ranked using a personalization score and visualized
through an interactive Power BI dashboard.

## Solution
Foodoscope provides:
- Personalized meal recommendations
- Filters based on cuisine, vegan preference, calories, and protein
- Clear visual insights for health and flavor balance
- Explainable ranking through a personalization score

## Dataset
The dataset contains vegetarian recipes with:
- Calories
- Protein
- Cuisine
- Flavor group
- Vegan flag

## Tools Used
- Microsoft Excel (data preparation & scoring)
- Power BI (dashboard & visualization)
